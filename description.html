Руководство SmartCalc by Egor O.

    В данном проекте реализован калькулятор с графическим интерфейсом. Он поддерживает как стандартные математические операции, так и вычисление тригонометрических и других функций, построение графиков данных функций, вычисление значения функции в конкретной точке. Для вычислений в калькуляторе используется алгоритм обратной польской нотации, вся графика же написана на Qt.

    • Основное окно:
      1. На основной странице располагаются числа, операции, функции, специальный символ 'x' и кнопка для построения графиков.
      2. Числа 0-9 и двойной нуль отвечают за ввод соответствующего значения.
      3. AC (all clear) - стирает всю входную строку.
      4. СE (clear element) - удаляет последний символ во входной строке.
      5. По нажатию на "graph" – откроется диалоговое окно для построения графиков.
      6. Если функция содержит x, пользователю будет предоставлена возможность подстановки нужного значения вместо x.
      7. e, символ для экспоненциальной формы записи.
      8. pi, при нажатии на которую подставляется 3.14...

    • Поддерживаемые операции:
      1. Вычисление операций со скобками: (a + b)
      2. Сложение: a + b
      3. Вычитание: a - b
      4. Умножение: a * b
      5. Деление: a / b
      6. Возведение в степень: a ^ b
      7. Остаток от деления: a mod b

    • Поддерживаемые функции:
      1. Косинус: cos(x)
      2. Синус: sin(x)
      3. Тангенс: tan(x)
      4. Арк косинус: acos(x)
      5. Арк синус: asin(x)
      6. Арк тангенс: atan(x)
      7. Квадратный корень: sqrt(x)
      8. Натуральный логарифм: ln(x)
      9. Десятичный логарифм: log(x)

    • Корректность ввода:
      1. Количество открывающих и закрывающих скобок - совпадает. (2+3)*2 - корректная запись, ((2+3)*2 - некорректная запись.
      2. Аргументы всех функций обозначены в скобках. sin(x) - корректная запись, sinx - некорректная.
      3. В записи выражения отсутствуют непредусмотренные литералы.
      4. При некорректном вводе – приложение сообщит об ошибке.
